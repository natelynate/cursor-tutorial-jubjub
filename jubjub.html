실행 방법: 프로젝트 루트에서 .env.example을 복사해 .env를 만들고 OPENAI_API_KEY를 넣은 뒤, npm install → npm start로 서버를 띄운 다음 이 페이지를 엽니다. API 키는 브라우저가 아니라 서버가 읽습니다.
